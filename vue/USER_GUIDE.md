## 普通用户功能

### 1.1 注册与登录

**注册**
1. 访问首页，点击右上角「注册」
2. 填写用户名、邮箱、密码
3. 点击「注册」完成注册

**登录**
1. 点击「登录」
2. 输入用户名和密码
3. 登录成功后可享受全部用户功能

### 1.2 浏览商品

**首页**
- 展示热门推荐商品
- 点击商品卡片查看详情

**商品列表**
1. 点击顶部导航「全部商品」
2. 可按分类筛选商品
3. 可设置价格区间筛选
4. 支持搜索商品名称
5. 支持按价格/时间排序
6. 支持分页浏览

**商品详情**
1. 点击商品进入详情页
2. 查看商品名称、价格、库存、描述
3. 选择购买数量
4. 查看相关推荐商品

### 1.3 购物车

**添加商品**
1. 在商品详情页选择数量
2. 点击「加入购物车」或「立即购买」

**管理购物车**
1. 点击顶部「购物车」进入
2. 可勾选商品进行批量操作
3. 可修改商品数量
4. 可删除商品
5. 查看商品总数和合计金额

**结算**
1. 确认选中的商品
2. 点击「结算」进入确认订单页

### 1.4 订单管理

**创建订单**
1. 在确认订单页填写收货地址
2. 可添加订单备注
3. 点击「提交订单」（提交时扣减库存，库存不足则创建失败）
4. 系统会发送确认邮件到用户邮箱
5. 用户点击邮件中的确认链接完成订单确认（24小时内有效，过期自动取消并恢复库存）
6. 确认成功后页面显示「订单确认成功」，可点击「查看订单详情」跳转至订单页
7. 如已打开订单详情页，请刷新页面查看最新状态（状态标签变为绿色的「已确认」）

**查看订单**
1. 点击「我的订单」查看所有订单
2. 可按状态筛选：全部/待确认/已确认/已完成/已取消

**取消订单**
1. 在订单列表找到「待确认」状态的订单
2. 点击「取消订单」
3. 订单状态变更为「已取消」，已扣减的库存自动恢复

### 1.5 个人中心

**查看个人信息**
- 查看用户名、邮箱、角色、注册时间等

**修改资料**
1. 进入「个人资料」页面
2. 修改邮箱等信息
3. 点击「保存修改」

**修改密码**
1. 进入「修改密码」页面
2. 输入当前密码和新密码
3. 点击「修改密码」
4. 修改成功后需重新登录

### 1.6 个性化推荐

**查看推荐**
1. 点击顶部用户名下拉菜单「个性化推荐」
2. 系统根据你的浏览和购买记录推荐商品
3. 可直接点击推荐商品进入详情页

## 销售人员功能

销售人员除了具备普通用户的全部功能外，还可进行以下操作：

### 2.1 进入管理后台

1. 登录后，点击用户名下拉菜单
2. 选择「管理后台」进入
3. 或直接在地址栏访问 `/admin`

### 2.2 商品管理

**查看商品列表**
- 可搜索商品
- 可分页浏览所有商品

**新增商品**
1. 进入「商品管理」页面
2. 点击「新增商品」
3. 填写商品名称、描述、价格、库存、分类、图片等信息
4. 设置商品状态（上架/下架）
5. 点击「确定」创建商品

**编辑商品**
1. 在商品列表点击「编辑」
2. 修改商品信息
3. 点击「确定」保存

**删除商品**
1. 在商品列表点击「删除」
2. 确认后商品被删除

### 2.3 订单管理

**查看订单**
1. 进入「订单管理」页面
2. 可按订单状态筛选
3. 查看订单详情，包括收货地址、商品信息、金额等

### 2.4 销售统计

**查看统计数据**
1. 进入「销售统计」页面
2. 可选择时间周期（日报/周报/月报）
3. 可选择日期范围
4. 查看关键指标：
   - 总销售额
   - 总订单数
   - 总用户数
   - 平均订单金额
5. 查看销售趋势表格

**触发推荐系统**
1. 在仪表盘点击「触发推荐」
2. 系统会重新计算用户个性化推荐

## 管理员功能

管理员具备系统最高权限，可进行以下操作：

### 3.1 用户管理

**查看用户列表**
1. 进入「用户管理」页面
2. 查看所有用户信息（ID、用户名、邮箱、角色、状态、注册时间）

**创建用户**
1. 点击「新增用户」
2. 填写用户名、邮箱、密码
3. 选择角色（普通用户/销售/管理员）
4. 点击「确定」创建

**删除用户**
1. 在用户列表点击「删除」
2. 确认后用户被删除

### 3.2 操作日志

**查看日志**
1. 进入「操作日志」页面
2. 查看所有系统操作记录
3. 包括：操作人、操作类型、目标类型、目标ID、操作时间

## 常见操作流程

### 购物流程

```
浏览商品 → 加入购物车 → 确认商品数量 → 结算 → 
填写收货地址 → 提交订单 → 邮箱确认 → 订单完成
```

### 商品管理流程（销售/管理员）

```
进入管理后台 → 商品管理 → 新增/编辑/删除商品
```

### 订单处理流程（销售/管理员）

```
进入管理后台 → 订单管理 → 查看订单详情 → 处理订单
```

### 推荐系统流程（管理员）

```
进入管理后台 → 仪表盘 → 触发推荐 → 系统重新计算推荐结果
```

## 安全说明

1. **密码安全**：密码使用加密存储，不会明文显示
2. **权限控制**：不同角色只能访问对应权限的页面和功能
3. **请求限流**：系统对频繁请求进行限制，防止恶意访问
4. **IP 封禁**：对异常行为进行 IP 级别的封禁
